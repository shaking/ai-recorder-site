技术支持

常见问题

Q：为什么没有生成摘要？

A：请在"设置 > AI 服务"中配置 AI 供应商的 API Key。未配置 API Key 时，仅支持本地录音和实时转写功能。

Q：支持哪些 AI 服务？

A：AI 摘要支持阿里云百炼、DeepSeek、Claude、OpenAI、Google Gemini、智谱 GLM、月之暗面 Kimi、字节豆包、MiniMax、OpenRouter。云端语音识别支持阿里云百炼、科大讯飞、腾讯云、Google Speech。

Q：如何获取 API Key？

A：在对应供应商的官网注册账号并创建 API Key。大部分供应商提供免费额度。在"设置"中选择供应商后，底部会显示对应的获取地址。

Q：录音数据会上传到云端吗？

A：所有录音和数据存储在您的本地设备。仅当您主动使用云端转写或 AI 摘要功能时，数据会发送到您配置的第三方服务。

Q：如何导出会议记录？

A：在会议详情页，点击右上角菜单，选择导出为 PDF、Word 或 Markdown。

Q：支持导入已有的音频文件吗？

A：支持。在会议库页面点击右上角菜单，选择"导入音频"，支持 M4A、MP3、WAV 等常见格式。导入后会自动进行云端转写和 AI 摘要。

Q：换手机后数据怎么迁移？

A：使用 iPhone 整机迁移或 iCloud 备份恢复，App 数据会自动迁移到新设备。

Q：摘要模板有什么区别？

A：每种模板针对不同场景优化输出结构。例如"项目进度"会用红黄绿状态灯标注每项工作的健康度，"客户沟通"会分析客户情绪和追踪承诺清单，"面试"会给出能力评分和录用建议。
