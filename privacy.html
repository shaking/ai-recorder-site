隐私政策

最后更新：2026 年 4 月

AI Recorder（以下简称"本应用"）尊重并保护用户的隐私。本隐私政策说明了本应用如何收集、使用和保护您的信息。

1. 数据收集与存储

本应用收集以下数据，所有数据均存储在您的本地设备上：

  • 音频录音：您使用本应用录制的会议音频
  • 转写文本：语音识别生成的文字内容
  • 会议摘要：AI 生成的摘要和分析结果
  • 地理位置（可选）：录音时的地理位置信息，仅在您授权后获取

本应用不设用户账号系统，不在我方服务器存储任何用户数据。

2. 第三方服务

本应用支持用户自行配置第三方 AI 服务，包括但不限于：

  • 语音识别服务：阿里云、科大讯飞、腾讯云、Google Speech
  • AI 摘要服务：DeepSeek、阿里云百炼、Claude、Google Gemini 等

当您使用云端语音识别或 AI 摘要功能时，录音音频或转写文本会发送至您选择的第三方服务进行处理。这些服务的数据处理受其各自隐私政策约束。

本应用不代替您向第三方服务发送数据，所有 API 调用使用您自己配置的 API Key，本应用不存储、收集或转发您的 API Key。

3. 权限使用

  • 麦克风：用于录制会议音频，仅在您主动开始录音时使用
  • 语音识别：用于实时语音转文字，仅在录音过程中使用
  • 位置信息（使用时）：用于记录录音的地理位置，可在系统设置中关闭

4. 数据安全

所有数据存储在您的 iOS 设备本地，受 iOS 系统沙盒保护。本应用不包含任何分析 SDK、广告 SDK 或用户追踪工具。

5. 儿童隐私

本应用不面向 13 岁以下儿童，不会故意收集儿童的个人信息。

6. 隐私政策更新

本隐私政策可能会不时更新。更新后的政策将在应用内或本页面公布。
